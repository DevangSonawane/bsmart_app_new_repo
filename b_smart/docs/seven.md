Create (+) – Step-by-Step User Flow
 
Step 0: Entry Point
1. User taps Create (+) in the Footer Navigation Bar.
2. App opens the Create Page in full-screen.
3. Footer navigation is hidden.
4. Top header shows two tabs:
○ Create New (default)
○ Upload
 
Step 1: Tab Selection
1. User sees Create New tab active.
2. User can swipe left/right or tap on the Upload tab to switch.
3. If unsaved content exists and user switches tabs, show discard warning.
 
Step 2A: Create New – Camera-Based Creation
Step 2A.1: Camera Interface
1. User sees full-screen camera preview.
2. Camera defaults to rear camera, can toggle to front camera.
3. Camera controls available:
○ Capture photo (tap)
○ Record video (tap & hold)
○ Flash toggle
○ Timer (optional)
○ Zoom pinch
○ Tap to focus
 
Step 2A.2: Apply Filters & Effects
1. User scrolls horizontal filter strip at the bottom.
2. Real-time preview updates as filters are applied.
3. Options include:
○ Color filters
○ Beauty filters
○ AR effects
○ Background blur/replacement
○ Motion effects (for video)
 
Step 2A.3: AI Enhancements
1. User taps AI features panel.
2. Selectable options:
○ Background removal / replacement
○ Face enhancement
○ Auto-crop & framing
○ Object detection
○ Caption suggestion
○ Video stabilization
○ Highlight reel creation
3. System processes media → shows preview.
4. User can accept or revert changes.
 
Step 2A.4: Add Background Sounds / Music
1. User taps Add Music/Sound.
2. Options:
○ Choose from existing platform library (trending sounds, tracks)
○ Extract audio from other videos
○ Record your own audio
3. User adjusts volume & sync with video.
4. Changes previewed in real-time.
 
Step 2A.5: Capture Completion & Edit Preview
1. User finishes capture.
2. Opens Edit Preview Screen:
○ Apply filters/AI enhancements
○ Trim video
○ Add/edit music
○ Adjust audio balance
3. When satisfied, tap Next → Post Details.
 
Step 2B: Upload – Gallery-Based Creation
Step 2B.1: Media Selection
1. User taps Upload tab.
2. Gallery grid opens.
3. User selects images/videos (single or multiple).
4. System validates supported media types.
 
Step 2B.2: Editing Tools
1. For images:
○ Crop / Rotate
○ Brightness / Contrast
○ Filters
○ Stickers / Text overlays
○ AI enhancements (cleanup, background removal, framing)
2. For videos:
○ Trim / Split
○ Filters
○ Music overlay
○ Speed control
○ AI auto-edit / thumbnail generation
 
Step 2B.3: Background Sounds / Music
1. Tap Add Music/Sound.
2. Options same as camera flow:
○ Library sounds
○ Extract audio from other videos
○ Record audio
3. Adjust volume & sync.
 
Step 2B.4: Upload Completion & Next
1. Tap Next → Post Details.
2. System validates file format, size, and length.
3. Proceed to Post Details.
 
Step 3: Post Details & Social Controls
1. User sees Post Details Screen:
○ Add caption
○ Add hashtags (#tags)
○ Tag friends
○ Set privacy: Public / Followers / Private
○ Enable/Disable comments
○ Add location
○ Optional cross-post sharing
2. User reviews all settings.
3. Tap Post / Publish → content is uploaded (flow beyond scope).
 
Step 4: Common Controls & Gestures
● Cancel → discard media, exit Create Page (with confirmation).
● Swipe down → close Create Page (discard confirmation).
● Pinch → zoom camera.
● Tap → focus.
● Long press → record video.
 
Step 5: Error & Edge Handling
1. Camera/gallery access denied → show permission CTA.
2. AI processing fails → option to retry or skip.
3. Unsupported file formats → error message.
4. Audio import fails → retry/skip audio.
 
✅ Summary of Flow:

● Entry: Tap Create (+)
● Choose Tab: Create New (camera) or Upload (gallery)
● Capture / Upload media
● Apply filters, AI enhancements, background sounds
● Edit media (trim, crop, add stickers/text/music)
● Tap Next → Post Details
● Add caption, tags, privacy, comment control, location
● Tap Post → content published
